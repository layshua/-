# 表格
- [x] table-enhancer：表格编辑 [Stardusten/ob-table-enhancer: Manipulate markdown tables without touching the source code in Obsidian. (github.com)](https://github.com/Stardusten/ob-table-enhancer)
- [x] Excel to Markdown Table：从 excel 钟复制表格
迁移表格的方法：将表格复制到 typora，进入源代码模式，再复制到 obsidian
- [x] css 片段：TableStyles [Custom CSS for tables](https://forum.obsidian.md/t/custom-css-for-tables-5-new-styles-ready-to-use-in-your-notes/17084)
- 可选风格：flatBlue/academia, academia-rounded（rounded 开启圆角），whiteRed, whiteRed-rounded/
- wideTable：全宽表格 
- leftAlign：左对齐 
- 使用格式：放在 frontmatter 中
```
---
   cssclass: purpleRed, wideTable, fixedFc
---
```
